\# Лабораторная работа №6: SciencePortalMVC with Web API



\*\*Студент:\*\* Сеноженский В.В.  

\*\*Группа:\*\* ИТП-31  

\*\*Вариант:\*\* 16



\## Описание



Веб-приложение на ASP.NET Core MVC и Web API для управления научным порталом. Включает в себя:

\- Управление кафедрами, преподавателями, проектами и публикациями.

\- Аутентификацию и авторизацию на основе ролей (Admin, User).

\- RESTful Web API для сущности "Проект".

\- Клиентское HTML-приложение для взаимодействия с API.

\- Настроенный CI/CD процесс с помощью GitHub Actions для автоматической сборки и тестирования.
